# Sistema de Análise Financeira de Vendas

## Descrição

Projeto desenvolvido em Python para análise de dados financeiros utilizando bases em Excel.

O sistema realiza a leitura dos dados, auditoria da qualidade das informações, cálculos de indicadores financeiros (KPIs), rankings de desempenho e geração automática de relatórios em Excel.

## Funcionalidades

- Leitura automática de arquivos Excel
- Auditoria da qualidade da base de dados
- Identificação de valores nulos
- Identificação de registros duplicados
- Identificação de inconsistências
- Cálculo de KPIs financeiros
- Ranking de produtos
- Ranking de vendedores
- Ranking de clientes
- Ranking por estado
- Análise por forma de pagamento
- Geração automática de relatório Excel

## Tecnologias

- Python
- Pandas
- OpenPyXL
- Excel
